ln.py
=====

You always get ``ln`` arguments in the wrong order?

Then ``ln.py`` is for you.

Installation
-------------

``pip3 install ln.py``

API
---

Instead of cryptic ``os.symlink(src, dest)``, use :
``ln.ln(from_=first, to=second)``
